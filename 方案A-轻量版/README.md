# 方案A：轻量版 — 最小可行产品

## 概述

纯搜索 + 安装功能。基于一个远端 JSON 精选库索引，配合 `npx skills find` 作为 fallback。

## 核心功能

1. **精选库搜索** — 从远端 JSON 拉取精选推荐列表，按关键词匹配
2. **扩展搜索** — 精选库无结果时，调用 `npx skills find` 搜索公开生态
3. **安全扫描** — 7 条安全红线检查
4. **一键安装** — 用户确认后执行 `npx skills add`

## 优势

- **实现成本最低**：只需维护一个 JSON 文件 + 一个 SKILL.md
- **立即可用**：今天就能部署上线
- **稳定可靠**：逻辑简单，不容易出错
- **体积最小**：单文件 Skill，上下文占用少

## 劣势

- **无能力缺口检测**：不能在用户使用其他 Skill 时自动推荐
- **无交叉导流**：卫星 Skill 无法反向引流到技能宝
- **推荐被动**：只有用户主动搜索才能触发
- **增长受限**：完全依赖用户主动搜索，缺少自然增长引擎

## 适用场景

- 快速验证技能宝概念
- 作为 MVP 版本上线测试
- 资源有限时的最优选择

## 依赖

- WebFetch（获取远端 JSON）
- Bash（执行 npx skills 命令）
