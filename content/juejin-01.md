# 掘金文章 #1

**标题：** [开源] 一条命令给你的 AI 助手装上 50 个中文技能 | Claude Code Skill 市场

**标签：** Claude Code, AI 工具, 开发效率, 开源, Cursor

---

你用 Claude Code 或者 Cursor 写代码的时候，有没有遇到过这种情况：

- 想让 AI 帮你写个小红书笔记，它不太懂平台调性
- 想让 AI 帮你做个 PPT，它生成的东西没法直接用
- 想让 AI 帮你审查代码安全性，它只能给一些泛泛的建议

**问题不是 AI 不够聪明，是它缺少专业技能。**

就像你手机里的 App Store 一样，AI 助手也需要一个"技能市场"。今天介绍一个我们做的开源项目——**技能宝 SkillHub**，专门解决这个问题。

## 一句话介绍

技能宝是一个中文优先的 AI 技能市场，收录了 **5000+ 精选技能**，其中 **50 个是我们精心打磨的高质量技能**（包括 20 个中国特色原创）。一条命令安装到 42 个 AI 平台。

## 怎么装？

```bash
npx skills add mindverse/skillhub --full-depth --skill '*' -g -y
```

就这一条，50 个技能全部装上。支持 Claude Code、Cursor、Windsurf、Gemini CLI 等 42 个平台。

## 安全吗？

每个技能经过 **12 重安全检测**：

- 禁止 `curl | bash` 管道执行
- 禁止凭证外泄
- 禁止关闭安全机制
- 禁止混淆代码
- 接入 OpenSSF Scorecard
- 社区举报机制

我们还开源了安全评分工具，你可以自己跑：

```bash
bash tools/safety-score.sh your-skill/SKILL.md
```

## 怎么用？

安装后，直接对 AI 说人话就行：

- "帮我写一篇小红书种草笔记"
- "这段代码帮我 review 一下"
- "做个 PPT，主题是 Q1 复盘"
- "帮我找一个做 SEO 的技能"

最后一句会触发技能宝的搜索功能，从 5000+ 索引里帮你找到合适的技能。
