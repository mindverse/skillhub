# V2EX 帖子 #1

**标题：** 做了一个中文 AI 技能市场，50 个技能一条命令装上（开源免费）

**节点：** /go/share

---

做了一个开源项目，给 Claude Code / Cursor 等 AI 助手做了个"应用商店"。

**一句话：** 一条命令安装 50 个中文技能，支持 42 个 AI 平台。

```
npx skills add mindverse/skillhub --full-depth --skill '*' -g -y
```

**解决什么问题：**

AI 助手很聪明，但缺少专业技能。比如你想让它写小红书笔记、做 PPT、审查代码安全，它做得不太好——不是不够聪明，是没有针对性的知识。

技能宝就是给 AI 装专业技能的。类似手机的 App Store，但装的不是 App，是 AI 的专业能力。

**有什么：**

- 50 个精选技能（20 个中国特色原创：小红书/抖音/公众号/知乎/B站/微信小程序/合同审查/高考作文/去AI味...）
- 5000+ 精选索引
- 12 重安全检测
- 支持 Claude Code、Cursor、Windsurf、Gemini CLI 等 42 个平台
- 完全免费，MIT 开源

**和腾讯 SkillHub 的区别：**

腾讯 3 月 10 号也出了一个，但他们内容以英文为主，安全审计不透明，不支持原创发布。我们做的是中文优先、安全评分可视化、去中心化分发。
