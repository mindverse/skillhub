昨晚折腾 Claude Code 的时候发现个东西，有点上头

你们用 Cursor 或者 Claude Code 吧？我之前一直觉得它就是个写代码的，直到我发现它居然能「装技能」

啥意思呢，就是你可以给 AI 加专门的能力包。比如你装上小红书的技能包之后，跟它说"帮我写个种草笔记"，它写出来的东西是真的有小红书味道的那种——不是那种一看就 AI 生成的干巴巴文案

终端里跑这一行就行了：

npx skills add mindverse/skillhub --full-depth --skill '*' -g -y

一口气给你装 50 个技能，我挑几个我实际试过的说说

写小红书——emoji 段落标签标题它全给你安排好，改改就能发
做 PPT——生成一个 HTML 文件，浏览器打开直接就是幻灯片
代码安全审查——OWASP 那些检查项它帮你一个个过

还有一堆离谱的：什么帮你写抖音脚本、写公众号、甚至写高考作文（？）合同审查也有，我试了一份租房合同，它标出来三个霸王条款，确实是

免费的，代码全开源。而且最重要一点：全中文，别的 skill 市场清一色英文，这个是真的给中文用户做的
