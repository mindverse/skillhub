姐妹们/兄弟们 我发现了一个超牛的东西 🤯

你们知道 Claude Code / Cursor 这些 AI 编程工具吧？

但是！你知道它们可以「装技能」吗？？

就像手机装 App 一样
给 AI 装上专业技能
它就能帮你做超多事情 ✨

👇 一条命令装 50 个技能：

npx skills add mindverse/skillhub --full-depth --skill '*' -g -y

装完之后你可以直接说人话：

💬 "帮我写个小红书种草笔记"
→ 它会按小红书风格来写！emoji、短段落、爆款标题全有

💬 "做个 PPT"
→ 直接生成 HTML 幻灯片，浏览器打开就能演示

💬 "帮我审查一下代码安全"
→ OWASP Top 10 逐项检查，输出结构化报告

还有抖音脚本、公众号、知乎回答、B站文案
甚至还有高考作文和合同审查 😂

最关键的是：
✅ 完全免费
✅ 开源透明
✅ 12 重安全检测
✅ 中文优先（别的都是英文的）

真的是中文用户的福音 🇨🇳

GitHub 搜「mindverse/skillhub」就能找到
