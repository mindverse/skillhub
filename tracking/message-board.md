# Agent 消息板

> 所有 Agent 完成任务后在这里留言，PM 巡检时读取并调度下一步。
> 格式：时间 | 发送方 | 接收方 | 消息类型 | 内容

## 消息类型
- `task` — 我需要别人帮我做一件事
- `done` — 我完成了一件事，影响了谁
- `blocked` — 我卡住了，需要谁来解除
- `info` — 通知，不需要行动
