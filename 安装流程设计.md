## 一、用户首次安装技能宝

### 1.1 安装命令

**Claude Code 安装方式：**

```bash
/install 技能宝 github:mindverse/skillhub
```


## 五、技术实现要点

### 5.1 安装机制

技能宝作为 Claude Code Skill，安装本质上是将 Skill 的 MD 文件写入到用户的 Skill 配置目录中。具体实现：

```
安装 = 将 SKILL.md 从 GitHub 拉取到本地 Skill 目录
卸载 = 从本地 Skill 目录删除对应文件
更新 = 拉取新版 SKILL.md 覆盖旧版
```

### 5.2 搜索机制

```
用户输入搜索词
       ↓
技能宝通过 WebFetch 拉取 skillhub-index.json
       ↓
在 Agent 上下文中对 JSON 数据进行语义匹配
       ↓
按推荐权重排序，返回结果
```

不需要复杂的后端搜索引擎——AI Agent 本身就是最好的语义搜索引擎。只需要给它结构化的数据，它能自动匹配用户的自然语言需求。

### 5.3 安全展示

安装前展示的安全信息来自 `skillhub-index.json` 中的 `safety_score` 和 `safety_details`，让用户感知到安全保障：

```
安全评分 95/100
├── 无数据收集 ✓
├── 代码审核通过 ✓
├── 外部请求已声明 ✓（仅访问 api.xiaohongshu.com）
└── 社区反馈正面 ✓（0 投诉）
```

### 5.4 离线降级

当远端索引无法访问时，技能宝使用内置精选库（硬编码在 SKILL.md 中的 Top 50 技能列表）进行推荐，确保基本功能不受网络影响。

## 六、关键设计原则

1. **路径要短**：从需求到安装完成，最少 2 步（搜索 → 确认安装）
2. **自然语言驱动**：不需要用户记住任何命令格式，说人话就行
3. **安全感先行**：每次安装前展示安全评分，让用户放心
4. **不打断工作流**：能力缺口推荐在任务间隙出现，不中断当前任务
5. **安装即可用**：安装完成后立即给出使用示例，降低学习成本
6. **不收集数据**：技能宝本身不收集用户行为数据，不需要登录注册
